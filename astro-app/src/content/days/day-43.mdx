(4) <strong>Memory</strong> — it persists context across sessions using project-level memory files.</p>\n  <p><strong>SDK/API access and GitHub Actions integration.</strong> Claude Code is available programmatically through the SDK and API, enabling integration into CI/CD pipelines and automated workflows.
